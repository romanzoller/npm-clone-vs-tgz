# npm-clone-vs-tgz

Showing that installing an npm package from

```
https://api.github.com/repos/<user>/<repo>/tarball/<hash>
```

is much faster than installing it from 

```
git://github.com/<user>/<repo>.git#<hash>
```

## Run

```
./run.sh
```
